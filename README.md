## ✨ Features

| Feature | Details |
|---|---|
| 🏋️ **Custom Habits** | Add any habit you want to track |
| 📅 **Monthly Grid** | Days across top, habits down side — tap to mark done |
| 🔥 **Auto Streak Counter** | Automatic streak per habit, updates daily |
| 📝 **Daily Diary** | Write notes saved per date |
| 📴 **Works Offline** | Full PWA — install to home screen |
| 🔒 **Privacy First** | All data stored locally — no server, no account |
| 🗑️ **Reset All** | One click to reset everything to zero |
| 🌙 **Dark UI** | Sleek dark purple/cyan theme |

---

## 🚀 How to Use

```
1. Click "+ Add Habit" to add your daily goals
2. Tap any cell in the grid to mark it done ✓
3. Watch your streak counter grow 🔥
4. Write your daily diary entry and save it
5. Install as PWA — works like a phone app!
```

---

## 📱 Install as App

```
Open live link → Browser menu → Add to Home Screen
Works like a native app icon on your phone! 📲
```

```bash

cd trackflow
# Just open index.html in your browser!
```

## ⚠️ Data Note

> All data is saved in `localStorage` in your browser. Clearing browser data will erase your history — don't clear site data if you want to keep your streaks! 💾
